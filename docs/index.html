    Welcome¶

    Welcome to the documentation for the angelnu Helm charts project.

    Getting started¶

    Helm must be installed to use the charts in this repository.

    Refer to Helm's documentation to get started.

    Installation¶

    helm repo add angelnu http://angelnu.github.io/helm-charts/

    You can then run helm search repo angelnu to search the charts.
